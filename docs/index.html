Diagnostic visualization for causal estimands

CP plots and local CP plots in a few lines of R.

CPplot is a plot-first R package for visualizing the relationship between estimated treatment effects and estimated propensity scores. It supports observational CP plots and local CP plots for instrumental-variable studies.

Start the tutorial View function map

Installation

Install the development version from GitHub and load the package.

install.packages("remotes")
remotes::install_github("e9tian/CPplot")
library(CPplot)

Workflow

CPplot separates estimation from visualization. You can estimate the inputs with any model you like, then pass the resulting columns to the plotting functions.

Estimate inputs

Compute propensity scores and conditional treatment-effect summaries.

Draw the plot

Use cp_plot() or local_cp_plot() for diagnostic graphics.

Read slopes

Use the returned slope table as a compact covariance diagnostic.

The plots are descriptive diagnostics, not formal tests of monotonicity. They are most useful over the part of the propensity-score support where the data are well represented.

Observational CP plot

In an observational study, the CP plot compares unit-level estimated CATEs tau_hat with estimated propensity scores ehat. The three fitted lines use full-sample weights 1, ehat, and 1 - ehat.

Minimal example

library(CPplot)

set.seed(1)
n <- 800
df <- data.frame(x1 = rnorm(n), x2 = rnorm(n))
df$ehat <- plogis(-0.4 + 1.1 * df$x1)
df$z <- rbinom(n, 1, df$ehat)
df$tau_hat <- 0.2 + 2.0 * df$ehat +
  1.3 * df$x2 + rnorm(n, sd = 0.35)

fit <- cp_plot(
  df,
  ehat = "ehat",
  tau_hat = "tau_hat",
  treat = "z"
)

fit$plot
fit$slopes
Returned object Meaning
fit$plot The ggplot object, ready to print or customize.
fit$slopes Slopes, standard errors, and p-values for the three diagnostic fits.
fit$data_used The finite observations retained after filtering missing or nonfinite inputs.

Local CP plot for IV studies

In an IV study, replace the CATE by an estimated conditional complier treatment effect. Under the IV assumptions used in the paper, pi_c_hat can be the estimated conditional first stage.

Minimal example

library(CPplot)

set.seed(2)
n <- 800
df <- data.frame(x1 = rnorm(n), x2 = rnorm(n))
df$ehat <- plogis(-0.3 + 1.0 * df$x1)
df$z <- rbinom(n, 1, df$ehat)
df$pi_c_hat <- pmax(
  0.05,
  plogis(-0.2 + 0.7 * df$x1 - 0.3 * df$x2)
)
df$tau_c_hat <- 0.4 + 2.4 * df$ehat +
  1.5 * df$x2 + rnorm(n, sd = 0.45)

fit <- local_cp_plot(
  df,
  ehat = "ehat",
  tau_c_hat = "tau_c_hat",
  pi_c_hat = "pi_c_hat",
  group = "z"
)

fit$plot
fit$slopes

Estimation helpers

CPplot includes simple transparent helpers for examples and quick checks. They use logistic regression for propensity scores and linear models for conditional contrasts. 

df2 <- estimate_cp_inputs(
  df,
  y = "y",
  z = "z",
  x = c("x1", "x2")
)

fit <- cp_plot(
  df2,
  ehat = "ehat",
  tau_hat = "tau_hat",
  treat = "z"
)

These helpers are optional. For applied work, you can estimate the inputs with causal forests, BART, SuperLearner, xgboost, or other models.

Paper demo data

The package bundles intermediate outputs used to reproduce the paper demo plots. These are estimated propensity scores and treatment-effect summaries, not the full raw-data analysis workflow. 

library(CPplot)

paper_datasets()[, c("name", "type", "loader")]

rhc <- load_paper_cp_data("rhc")
rhc_treat_col <- attr(rhc, "treatment_column")
head(rhc[, c("propensity_scores", "tau_hat", rhc_treat_col)])

k401 <- load_paper_401k_data()
head(k401[, c("ehat", "Z", "delta_d", "tau_c_hat")])

Function map

cp_plot()

Draw an observational CP plot from ehat, tau_hat, and optional treatment labels.

local_cp_plot()

Draw a local CP plot from IV propensity scores, complier effects, and complier weights.

cp_slopes()

Return the three full-sample weighted slope diagnostics for an observational CP plot.

local_cp_slopes()

Return the complier-weighted slope diagnostics for a local CP plot.

estimate_cp_inputs()

Fit simple example propensity-score and outcome models for observational examples.

estimate_local_cp_inputs()

Fit simple example IV propensity, reduced-form, and first-stage models.
